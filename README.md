# Dashboard Map + Timeline App

This app visualizes time-based geospatial data on a map with an interactive slider.

## Stack
- Next.js + TypeScript
- Zustand
- Mapbox GL JS
- Material UI (Slider)
- Recharts (Optional)

## Setup

```bash
npm install
```

Create `.env.local`:

```
NEXT_PUBLIC_MAPBOX_TOKEN=your_token_here
```

Run the app:

```bash
npm run dev
```
